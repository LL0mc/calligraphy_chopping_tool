# 前端重设计计划 — Digital Ink（数字墨韵）

## 审美方向

**Digital Ink** = 文人书卷气 × 科技质感。保留书法工具的文化内核，用现代 UI 语言重新表达：

- 基底：深色（`#0c0c14`）如墨池，带微蓝紫底光
- 表面：玻璃拟态（`backdrop-filter: blur`）+ 微噪点 grain
- Accent：朱砂红发光 `#e8453c`（传统）+ 电光蓝 `#4a7cf7`（科技）
- 字体：ZCOOL QingKe HuangYou（标题书法感）+ Noto Sans SC（UI 现代感）+ JetBrains Mono（数据等宽）
- 动效：staggered reveal、发光脉冲、hover 微反馈

---

## 实施原则

- **不改后端逻辑** — 不碰 Python API、数据模型、路由
- **不改前端交互逻辑** — 保持现有 JS 功能（拖拽、保存、提交等），只改表现层
- **增量实施** — 每次改一个页面，当前聚焦**校对页面**（分割场景）

---

## 阶段一：校对页面（review_server.py 内联 HTML）

当前 inline HTML 约 640 行，CSS 全内嵌在 `<style>`，JS 全内嵌在 `<script>`。

### 改动方案

#### 1. CSS 结构

| 当前 | 改为 |
|------|------|
| 全部内联在 `HTML = r"""..."""` 中 | 内联 `<style>` 保留（不增加文件依赖），但用 CSS 变量重构 |
| 无 CSS 变量，颜色散落各处 | 定义 `:root` 变量，统一主题 tokens |

```css
:root {
  --bg-primary: #0c0c14;
  --bg-surface: rgba(255,255,255,0.04);
  --bg-glass: rgba(255,255,255,0.06);
  --border-glass: rgba(255,255,255,0.08);
  --text-primary: #e8e0d4;
  --text-muted: #908078;
  --accent-red: #e8453c;
  --accent-blue: #4a7cf7;
  --glow-red: 0 0 12px rgba(232,69,60,0.4);
  --glow-blue: 0 0 12px rgba(74,124,247,0.3);
  --radius-sm: 6px;
  --radius-md: 10px;
  --radius-lg: 16px;
  --font-ui: 'Noto Sans SC', sans-serif;
  --font-display: 'ZCOOL QingKe HuangYou', cursive;
  --font-mono: 'JetBrains Mono', monospace;
  --transition: 0.2s ease;
}
```

- 左侧图片区：加玻璃质感边框（`border: 1px solid var(--border-glass)` + 微圆角 + hover 时 `box-shadow` 发光）
- 右侧面板：改为玻璃卡片组，每张卡片独立玻璃背景 + 圆角 + 微间距
- 状态标签、按钮组重新分组，视觉层次清晰

#### 3. 色彩具体变化

| 元素 | 当前 | 改为 |
|------|------|------|
| 页面背景 | `#1a1612` | `#0c0c14` + 径向渐变光晕 `radial-gradient(circle at 50% 0, rgba(74,124,247,0.06), transparent 60%)` |
| 工具栏 | 行内排列 | 毛玻璃底栏，半透明 + blur |
| 卡片/面板 | `#2a2520` 纯色 | `rgba(255,255,255,0.04)` + `backdrop-filter: blur(12px)` + 1px 微光边框 |
| 输入框 | `#3a3028` | 玻璃质感 `rgba(255,255,255,0.06)`，focus 时发光边框 |
| 按钮 | 纯色 background | hover 发光 + 微上浮 `translateY(-1px)` |
| 删除按钮 | `#5c2e2e` | 红色 glow，hover 增强 |
| 提交按钮 | `#6b5b4a` | 渐变背景，hover 发光 |
| 颜色标记（图例） | 纯色文字 | 发光圆点 + 标签组合 |
| 表格 | 无背景条纹 | 行 hover 发光条（左侧 2px 竖线） |
| 滚动条 | 默认 | 细滚动条，暗色 thumb，hover 时显示 |
| 加载遮罩 | `rgba(0,0,0,0.7)` | 毛玻璃 `backdrop-filter: blur(8px)`，spinner 更换 |

#### 4. Typography 变化

| 元素 | 当前 | 改为 |
|------|------|------|
| 页面标题 | 无 | 用 `--font-display` 显示"字帖校对" |
| 图例标签 | 11px 灰色 | 12px `--font-ui`，加 icon 圆点 |
| 表格文本 | 13px | 13px `--font-ui` + `--font-mono` 用于坐标数字 |
| 段落预览 | 14px | 15px `--font-ui`，行高 1.6 |

#### 5. 动效与微交互

| 场景 | 动效 |
|------|------|
| 页面加载 | 工具栏 + 左右面板 staggered fade-in（`animation-delay: 0.1s/0.2s/0.3s`） |
| 按钮 hover | `box-shadow` 发光增强 + `translateY(-1px)` 上浮，`transition: 0.2s` |
| 输入框 focus | 边框渐变为 accent 色 + 外层发光 |
| 表格行 hover | 左侧 2px 发光竖线（`::before` 伪元素滑动出现） |
| 选中行 | 左侧竖线常亮 + 背景微亮 |
| 切换字符 | 右侧面板信息微 fade 刷新 |
| 拖拽框 | canvas 上框线发光脉动 |
| 状态标签 | 圆点指示灯（呼吸动画） |
| 加载遮罩 | spinner 替换为墨滴涟漪 SVG 动画 |
| 图例颜色标记 | 小圆点 + 发光 |

#### 6. Canvas（检测框渲染）微调

当前 canvas 绘制在 `dc()` 函数中（纯 canvas 2D API，不用 Fabric.js）。不改绘制逻辑，只调整：

- 选中框由纯绿 `#00ff00` → 朱砂红发光 `#e8453c` + 微 glow 效果
- 控制点尺寸由 5px → 6px，带白色边框
- 字体标签由纯白 `#ffffff` → 半透明白 `rgba(255,255,255,0.85)`，字号 14px
- 标签位置保留 auto-flip 逻辑，但加微阴影提高可读性
